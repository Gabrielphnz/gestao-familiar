    Atalho iPhone 🍎

    Use o app Atalhos para registrar despesas com a Siri.

    1. Adicione este app à tela inicial (iOS 16.4+)

    2. Abra Atalhos → Novo Atalho

    3. Adicione "Pedir entrada" (valor e descrição)

    4. Ação "Obter conteúdo URL" → POST para Firebase

    5. Diga "Ei Siri, adicionar despesa"

    Dica: Instalar como PWA na tela inicial já resolve 90% dos casos — o app abre instantâneo, sem barra do browser.
