    Monitor Android 🤖

    Lê notificações bancárias (Nubank, Itaú, Bradesco…) e lança automaticamente. Grátis, sem root.

    # Cole no Termux (Android)

    pkg install -y python termux-api && pip install requests

    1. Instale Termux + Termux:API no F-Droid ou Play Store

    2. Cole o comando acima no Termux e execute

    3. Baixe o monitor: monitor.py (disponível no GitHub)

    4. Edite e cole seu UID e Token no topo do arquivo

    5. Execute: python monitor.py

    6. Conceda permissão de notificações quando solicitado

    Cole este comando no Termux após instalar o monitor — ele preenche UID e Token automaticamente.

    Atalho iPhone 🍎

    Registre despesas com a Siri ou um toque na tela inicial. iOS 16.4+.

    1. Abra o app Atalhos → toque em +

    2. Adicione: Pedir entrada → Tipo (Despesa/Receita)

    3. Adicione: Pedir entrada → Valor (número)

    4. Adicione: Pedir entrada → Descrição (texto)

    5. Adicione: Obter conteúdo URL → POST para:

    https://webhook-petxgwxf5a-uc.a.run.app

    6. No corpo JSON cole seu UID e Token (botão abaixo)

    7. Nomeie como "Adicionar Gasto" e adicione à tela inicial

    8. Diga: "Ei Siri, Adicionar Gasto"

    Dica rápida: Instalar como PWA na tela inicial já resolve 90% dos casos — abre instantâneo, sem barra do browser.
